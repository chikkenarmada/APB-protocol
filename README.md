# APB Protocol Verification

## project Overview
This project demonstrates the verification of AMBA APB (Advanced Peripheral Bus) protocol using SystemVerilog. The design validates read and write operations with proper handshake signals.

---

##  Features
- APB Write Operation Verification
- APB Read Operation Verification
- Signal-level analysis of:
  - PADDR
  - PWRITE
  - PSEL
  - PENABLE
  - PREADY

##  Verification Components
- APB Driver
- APB Monitor
- APB Agent
- Assertions
- Functional Coverage
